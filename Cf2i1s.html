문제

챗봇과 대화하는 문제입니다 실행순서를 클릭하세요 
안녕? 넌 이름이 뭐야: 재석
만나서 반가워, 재석. 
넌 주로 뭐하고 살아: 학교다녀
학교다녀? 나도 마찬가지야 ㅎㅎ 
그럼 심심할 때 뭐하니: 영화봐
영화봐? 나랑 똑같네. 
혹시 좋아하는 색은 뭐야 : 초록색
초록색이라구? 와~ 이것도 나랑 똑같네. 신기하다.
앞으로 우리 친하게 지내자.

프로그램 코드

	#include <stdio.h>								


1	int main() {

		char name[32] = "";

		char job[32] = "";

		char hobby[32] = "";

2		char color[32] = "";


3		printf( "안녕? 넌 이름이 뭐야: " );					

4		scanf( "%s", name );							

5		printf( "만나서 반가워, %s. ", name );					


6		printf( "넌 주로 뭐하고 살아: " );					

7		scanf( "%s", job );							

8		printf( "%s? 나도 마찬가지야 ㅎㅎ ", job );				


9		printf( "그럼 심심할 때 뭐하니: " );					

10		scanf( "%s", hobby );							

11		printf( "%s? 나랑 똑같네. ", hobby );					


12		printf( "혹시 좋아하는 색은 뭐야 : " );					

13		scanf( "%s", color );						

14		printf( "%s이라구? 와~ 이것도 나랑 똑같네. 신기하다. \n", color );	


15		printf( "앞으로 우리 친하게 지내자." );					


16		return 0;

	}
