문제

점수를 입력받아 그 값을 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
0점 ~ 100점 사이의 점수를 입력하세요: 89
당신의 점수는 89점입니다.


0점 ~ 100점 사이의 점수를 입력하세요: 74
당신의 점수는 74점입니다.


0점 ~ 100점 사이의 점수를 입력하세요: 95
당신의 점수는 95점입니다.

알고리즘

프로그램시작 
   사용자가 점수를 입력 
   사용자가 입력한 점수를 출력 
프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter02/Score.c 

	#include<stdio.h> 

	 

	// 프로그램시작 

1	int main() { 

		// 점수 초기화 

2		int score = 0; 

	 

		// 사용자가 점수를 입력 

3		printf( "0점 ~ 100점 사이의 점수를 입력하세요: " ); 

		// 89점을 입력한다고 가정 

4		scanf( "%d", &score ); 

	 

		// 사용자가 입력한 점수를 출력 

5		printf( "당신의 점수는 %d점입니다.\n", score ); 

		 

		// 프로그램 종료 

6		return 0; 

	}
